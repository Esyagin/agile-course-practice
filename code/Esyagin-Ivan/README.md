## Задание

Разработать приложение, записывающее список дел, необходимых сделать пользователю. Необходимо реализовать:

 - Добавление, удаление и редактирование дел
 - Вывод списка по степени важности и по дате (до какого момента дело еще актуально)
 - разделение дел по разным категориям (работа, учеба и т.д.)
